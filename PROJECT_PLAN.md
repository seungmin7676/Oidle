# 프로세스 스케줄링 시각화 프로젝트 계획서

## 프로젝트 개요

> **"유한한 CPU 연산 시간을 효율적으로 재활용하는 프로세스 스케줄링 과정의 시각화"**

운영체제(OS)에서 프로세스가 생성되고 소멸되는 과정은 '시간'과 '메모리'라는 자원을 소비하는 과정이다.  
이미 흘러가 버리면 사라지는 **유휴 시간(Idle Time)**을 스케줄링 알고리즘을 통해 유의미한 연산 시간으로 전환시키는 과정 자체가 디지털 자원의 선순환이자 재활용이다.  
본 프로젝트는 이 과정을 웹 기반으로 시각화하여, '유휴 시간' 없이 최적의 처리 성능을 구현하는 **디지털 자원 재생산**을 목표로 한다.

## 핵심 기능

### 1. 프로세스 생성 및 관리
- 사용자가 직접 가상 프로세스를 생성
- 입력 항목:
  - 프로세스 이름 (Process Name)
  - 도착 시간 (Arrival Time)
  - 실행 시간 (Burst Time)
  - 우선순위 (Priority) - 우선순위 스케줄링 사용 시
- 생성된 프로세스 목록을 테이블 형태로 표시
- 프로세스 삭제 및 수정 가능

### 2. 실시간 스케줄링 시뮬레이션
- 스케줄링 알고리즘 선택 후 '실행' 버튼으로 시작
- 지원 알고리즘 (확정):
  - **FCFS** (First Come, First Served) - 선입선출
  - **SJF Non-Preemptive** (Shortest Job First) - 비선점형 최단 작업 우선
  - **SRTF / SJF Preemptive** (Shortest Remaining Time First) - 선점형 최단 작업 우선
  - **Round Robin (RR)** - 라운드 로빈 (타임 퀀텀 설정 가능)
  - **Priority Non-Preemptive** - 비선점형 우선순위 스케줄링
  - **Priority Preemptive** - 선점형 우선순위 스케줄링
- 애니메이션을 통해 프로세스가 CPU를 점유하는 과정을 실시간으로 표현

### 3. 자원 점유 시각화
- **간트 차트 (Gantt Chart)**: 시간 축에 따라 각 프로세스의 CPU 점유 구간을 색상으로 구분하여 표시
- **CPU 이용률 게이지**: 현재 CPU가 유휴 상태인지 작업 중인지 실시간 표시
- **프로세스 상태 표시**: Ready / Running / Waiting / Terminated 상태 전환을 시각적으로 표현
- 각 프로세스에 고유 색상 부여

### 4. 성능 분석 대시보드
- 전체 프로세스 종료 후 자동 계산 및 출력
- 분석 항목:
  - 평균 대기 시간 (Average Waiting Time)
  - 평균 반환 시간 (Average Turnaround Time)
  - 평균 응답 시간 (Average Response Time)
  - 총 실행 시간 (Total Execution Time)
  - CPU 이용률 (CPU Utilization %)
  - 처리량 (Throughput)
- 알고리즘 간 성능 비교 기능 (선택적)

## 기술 스택 (확정)

| 구분 | 기술 |
|------|------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla JS) |
| **시각화** | Chart.js (CDN) - 간트 차트, 통계 차트 |
| **레이아웃** | Flexbox / Grid |
| **UI 테마** | 모던 그라데이션 테마 (컬러풀하고 세련된 느낌) |
| **빌드 도구** | 없음 (순수 HTML/CSS/JS + CDN 라이브러리) |

> Chart.js CDN을 통해 시각화 라이브러리를 활용하며, 그 외 핵심 로직은 순수 Vanilla JS로 구현

## 예상 프로젝트 구조

```
Project/
├── index.html              # 메인 페이지
├── css/
│   ├── style.css           # 전체 스타일 (그라데이션 테마)
│   ├── dashboard.css       # 대시보드 스타일
│   └── animations.css      # 애니메이션 효과
├── js/
│   ├── main.js             # 진입점, 전체 흐름 제어
│   ├── process.js          # 프로세스 생성/관리 로직
│   ├── scheduler.js        # 스케줄링 알고리즘 구현
│   ├── visualizer.js       # 간트 차트 및 시각화
│   └── dashboard.js        # 성능 분석 및 통계
└── assets/
    └── icons/              # 아이콘 및 이미지 리소스
```
